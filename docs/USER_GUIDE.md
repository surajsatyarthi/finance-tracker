# User Guide

## Navigation
- Dashboard: overview, KPIs, projections, reminders
- Transactions → Add: income/expense/transfer with recurring and partition
- Accounts: balances, transfers
- Loans: EMI schedule; mark EMI as paid
- Cards: limits, balances, utilization, statement day
- Savings & FDs: maturities, ladder
- Goals: targets, suggested monthly contribution, quick add
- Budget: inline editor, copy previous month, variance drilldown

## Workflows
- Add an expense: Transactions → Add → Expense → fill details → optional Recurring → Partition → Save
- Add an income: Transactions → Add → Income → source + partition → Save
- Transfer funds: Transactions → Add → Transfer → source/destination → Save
- Update account balance: Accounts → Edit balance (modal)
- Mark EMI paid: Loans → Mark Paid
- Update goal: Goals → type amount and press Enter
- Edit budget: Budget → Edit This Month Budget → set category values → blur to save

## Privacy Lock
- Header → Lock: set a passphrase; amounts mask while locked
- Auto-lock triggers after inactivity; unlock with passphrase

## Backups
- Encrypted export/import available in DataBackup (AES-GCM). Store backup safely.

## Tips
- Use Business/Personal partition consistently for clean dashboards
- Tag recurring bills to improve projections; mark EMIs for reminders
- Keep budgets realistic; copy previous month and tune top categories
